## Absolute imports

> To remove visual clutter of relative paths.

Instead of using relative import paths in each components

```javascript showLineNumbers
import User from '../../components/User';
```

Use absolute imports

```javascript showLineNumbers
import User from '@/components/User';
```

## tsconfig.json

Add **baseUrl** and **paths** options to your config file.

```json:tsconfig.json {3-6} showLineNumbers
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["*"]
    }
  }
}
```

Now you can use **'@/'** as prefix to path with **src** as your base folder.
